## Getting Started

Prerequisites
Node.js (latest LTS version recommended)
npm or yarn

## Installation
1. Clone the repository
``bash
git clone https://github.com/Pendekar-Naga/singalam-fe.git
`

2. Install dependencies
`bash
npm install
or
yarn install
`

3. Set up environment variables
`bash
cp .env.example .env
`

4. Start development server
`bash
npm run dev
or
yarn dev
`
